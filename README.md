
# QR Code Generator

Application built with Next.js and CSS to generate dynamic and customizable QR Codes. The user can enter a link, choose the code and background colors, upload an image to use as a logo in the center, and define the logo size. The QR Code is displayed in real time and can be downloaded as an image.
This project also includes unit tests implemented with Jest and React Testing Library to ensure component reliability and maintainability.

## Getting Started

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```
3. Start the development server:

```bash
npm run dev
```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the project.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
